# DayZ CF-Tools Cloud C# SDK


This SDK gives you the ability to post or query data from the CFTools Cloud API.
You can run anything available in the webapi through this SDK. Currently supported is:
- [x] #Fetch Leaderboard
- [ ] #Fetch Playerdata
- [ ] #Fetch Server data
